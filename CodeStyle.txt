*CodeStyle*

1. В начале объявляются переменные доступа "public", затем все остальное, отделять переменные по доступу переносом строки.
2. Название переменных доступа "private" начинается с "_".
3. Комментарии для 1 строки пишутся справа, на определенном растоянии, которое сохраняется на протяжении всего файла.
4. Комментарии для 2-ух и более строк пишутся над ними, после кода следует перенос строки.
5. Комментарии для функции, метода, или конструкции типа условия или цикла, пишутся над ней.
6. После ключевых слов (if, else, etc...) ставится "{" перенос строки, перенос строки, после пишется код.

using ...

public class ...{

	public float speed = 2.0f; 							//Cкорость

	private CharacterController _charController; 					//Переменная для ссылки на компонент типа CharacterController
	private Vector3 _movement = new Vector3();					//Создание вектора для будущей его передачи в Move
	
	//Координаты
	private float _x = 0f;
	private float _y = 1f;
	private float _z = 2f;

	//...
	void Start () {
		... 									//...
	}

	//Метод, который вызывается каждый кадр
	void Update () {
		
		...									//...
		...									//...
		...									//...
		...									//...
	
		...									//...
		...									//...
		...									//...
	}
}
